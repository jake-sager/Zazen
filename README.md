# Zazen

[Zazen is a meditative discipline that is typically the primary practice of the Zen Buddhist tradition.](https://en.wikipedia.org/wiki/Zazen)

The old meditation app I was using got super bloated. 

**Zazen** is simple: just a timer, intervals, and stats.
No ads. No upsells. Free. Simple.
